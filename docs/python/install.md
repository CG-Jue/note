#### 安装pip

##### 什么是pip

*pip 是一个现代的，通用的 Python 包管理工具。提供了对 Python 包的查找、下载、安装、卸载的功能。*

Python安装的时候选择Custom install可以直接安装

### **验证**

`Win键`+`R` 打开运行 输入cmd或powershell

输入`python --version` 显示python版本

输入`pip --version` 显示pip版本

就安装成功了
